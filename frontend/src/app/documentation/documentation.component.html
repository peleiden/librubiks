Documentation

This is a website for showing our results from our second year project at DTU (course 02466). It is intended to show our results in a way that allows everyone to try it out without needing to set anything up.

The site is hosted GitHub Pages and by default uses a Python backend hosted on Heroku, though it is possible to host your own local webserver if you want to utilize local CUDA availability. The repository is available on GitHub.

We used PyTorch 1.5 for implementing our learning algorithm in Python. The site is build using Angular 9 with Bootstrap 4 for styling.

Starting Your Own Local Backend

The free Heroku server we use has no CUDA and only 500 megabytes of memory, so the solvers are slow and prone to cause memory overflow errors if given too much time. You can start your own server by going through the following steps:

  1. Getting librubiks and dependencies, see the repo.
  2. Running pip install wget flask flask_cors flask_restful
  3. From the librubiks folder, run python librubiks/api.py
  4. Choose Local in the top right corner.

You will need a PyTorch compatible GPU to realize the full potential of the agents.
