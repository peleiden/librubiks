This is a website for showing our results from our second year project at DTU (course 02466). It is intended to show our results in a way that allows everyone to try it out without needing to set anything up. It also includes an extended appendix for things that are not practical to include in a report, such as the GIF for showing target value development, or interesting but less relevant things.

The site is hosted GitHub Pages and by default uses a Python backend hosted on Heroku, though it is possible to host your own local webserver if you want to utilize local CUDA availability. The repository is available on GitHub.

We used PyTorch 1.5 for implementing our learning algorithm in Python. The site is build using Angular 9 with Bootstrap 4 for styling.
